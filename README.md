# Money₂Time Web

Product landing page and privacy policy for the [Money₂Time](https://apps.apple.com) iOS app — a personal finance app that converts spending into hours of your life.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- DM Sans / DM Mono fonts

## Pages

- `/` — Landing page with hero, "Why I Built This" section, and download links
- `/privacy` — Privacy policy for App Store compliance

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Build

```bash
npm run build
npm start
```
